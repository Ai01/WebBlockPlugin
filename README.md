# WebBlockPlugin
webBlockPlugin

## finished feature

* block 页面
* 添加当前页面到block中
* 添加block网站的ui
* 删除block的网站
* icon
* 数据存储
* 重定向，对于被block的网站，可以选择重定向或者提示
* 在popup页面加上网页的信息
